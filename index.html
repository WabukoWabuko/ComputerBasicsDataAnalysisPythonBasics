Computer Basics

Understanding the machine from the ground up. No assumptions, just clear steps.

What is a Computer?

Imagine a Kitchen. You have a Chef (the brain), a countertop (the workspace), and a pantry (the storage). The computer works exactly the same way.

A computer is an electronic device that takes raw facts (Input), processes them into something useful, and shows you the result (Output).

The Hardware (The Physical Stuff)

CPU

The Chef. Does all the thinking.

RAM

The Countertop. Temporary workspace.

SSD/HDD

The Pantry. Where files live forever.

Files & Folders

Think of your computer as a giant digital filing cabinet.

The Path: Files live inside Folders. Folders can live inside other Folders.
Extensions: Every file has a "Last Name" that tells the computer what it is.

Workplace Essentials

Microsoft Office Suite

The digital toolkit for every professional office task.

üìó Microsoft Excel: The Number Cruncher

Excel is a grid of Cells. Each cell has an address, like B2 (Column B, Row 2).

Excel is just a very powerful calculator that remembers every number you typed.

Creating a Formula

Every formula MUST start with an Equals sign =. Without it, Excel thinks you are just typing text.

// Adding two cells together
=A1 + B1

// Using a Function (a built-in tool)
=SUM(A1:A10)
üìò Microsoft Word: The Writer

Focus on the Ribbon (the menu at the top). It is grouped by what you want to do:

  • Home: Changing how text looks (Bold, Size, Color).
  • Insert: Adding "Stuff" (Pictures, Tables, Page Numbers).
  • Layout: Changing the paper size or margins.
Common Beginner Mistake: Not "Saving As" immediately. Always name your file and save it to a folder you recognize before you start working!

Data Analysis Fundamentals

Turning raw numbers into stories and decisions.

The 4-Step Process

Ask: What problem are we trying to solve?
Clean: Remove duplicates and fix spelling errors in your data.
Analyze: Find the Average, the Highest, and the Lowest.
Share: Create a chart so others can see the trend.

Python Programming

Talking to the computer in a language it understands.

What is a Variable?

A Variable is like a labeled Box. You put a value inside the box so you can use it later.
# We label a box 'name' and put "Alice" inside
name = "Alice"
age = 25

print(name)

Logical Thinking (If/Else)

Programming is just making decisions based on rules.

if age > 18:
    print("You are an adult")
else:
    print("You are a minor")
Gentle Reminder: Computers are very literal. If you miss a single colon (:) or a quote mark ("), the code won't run. This is normal! Every programmer spends half their time fixing these small typos.
