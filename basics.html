The Kernel and the Shell

A "Guru" understands that the Windows you see is just a GUI (Graphical User Interface). Underneath sits the Kernel.

  • The Kernel: The manager. It decides which app gets to use the CPU or RAM.
  • The Shell: The interface that lets humans talk to the Kernel.

The Command Line Interface (CLI)

Gurus don't use folders to move files; they use the terminal for speed and automation.

> cd Desktop/Project_X
> mkdir assets scripts docs
> dir /s

Try explaining `mkdir` (Make Directory) and `cd` (Change Directory) to the student.

Networking & The Web Guru

How does the internet *actually* work? Explain the DNS (The Phonebook of the Internet).

Step-by-Step:
1. User types google.com.
2. DNS converts it to an IP Address (e.g., 142.250.190.46).
3. Your computer sends a Request; the server sends a Response (HTML).
